Introducing BlockHawk

Core Capabilities

  • Real-time Transaction monitoring with millisecond precision
  • Multi-chain Support for both Solana and Ripple
  • Statistical Anomaly detection using AI/ML techniques

How BlockHawk Detects Attacks: Real Examples

Example 1: Mango Markets ($114M Exploit)

During the October 2022 Mango Markets exploit, an attacker manipulated price oracles by creating artificial trading activity. BlockHawk would have detected this attack through:

  • Z-Score Analysis: Transactions of 96 SOL generated Z-scores of 156, far above normal thresholds
  • Time-Series Analysis: 400% spike in transaction volume minutes before the critical vulnerability was exploited
  • Frequency Pattern Detection: Abnormal burst of 12 transactions in under 2 minutes from a previously quiet address

Detection time: 38 seconds before critical fund loss

Example 2: Cashio ($52M Exploit)

In the March 2022 Cashio exploit, an attacker injected unauthorized accounts into the validation flow. BlockHawk would have detected this through:

  • Graph-Based Pattern Recognition: Identified unauthorized accounts in the transaction flow
  • Z-Score Analysis: Detected statistically significant deviations in transaction amounts
  • Adaptive Thresholding: Adjusted sensitivity based on the protocol's normal behavior patterns

Detection time: 45 seconds before critical fund loss
